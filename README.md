# 🌐 CryptoVerse - Web3 NFT Generator

> Transform Physics Simulations into Infinite NFTs

A modern, funky, and responsive Web3 frontpage featuring real-time 3D physics simulations, video recording, and blockchain NFT minting.

## ✨ Features

### 🎨 Visual Design
- **Neon Crypto Aesthetic**: Bold gradients, glitch effects, and glowing elements
- **Responsive Design**: Mobile-first approach with seamless scaling
- **Particle System**: Dynamic background particles with physics
- **Starfield Background**: Animated starfield effect

### 🔮 3D Physics Studio
- **Three.js 3D Rendering**: Advanced 3D scene with multiple lighting
- **Cannon.js Physics Engine**: Real-time physics simulation
- **Interactive Controls**:
  - Gravity adjustment (0-100)
  - Particle count (10-500)
  - Velocity control
  - Color palette selection

### 📹 Recording System
- Capture individual frames as PNG
- Video recording at 30 FPS

### 💰 Web3 & NFTs
- **MetaMask Integration**: Full wallet support
- **NFT Minting**: Create NFTs from simulations
- **Rarity Calculation**: Dynamic attributes
- **Gallery Display**: View your creations

## 🚀 Quick Start

```bash
git clone https://github.com/ch0senn/web3-nft-generator.git
cd web3-nft-generator
python -m http.server 8000
# Visit: http://localhost:8000
```

## 📦 Technologies
- Three.js - 3D rendering
- Cannon.js - Physics simulation
- Web3.js - Blockchain interaction
- MetaMask - Wallet connection
